Chrome Password Export Instructions - March 20, 2026
=================================================

Manual Steps Required:

1. Chrome is now open at chrome://settings/passwords
2. Click the three dots menu next to "Saved Passwords"
3. Select "Export passwords..."
4. Authenticate with macOS password when prompted
5. Save the CSV file to: ~/.openclaw/workspace/backups/chrome_passwords_2026-03-20.csv

Alternative method:
1. Open Chrome
2. Go to: chrome://password-manager/settings
3. Click "Download file" under Export passwords
4. Save to the backup directory

Note: This requires manual user interaction for security reasons.
Chrome does not allow programmatic password export.
