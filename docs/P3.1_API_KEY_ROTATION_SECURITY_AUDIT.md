## Security Audit Checklist

### Quarterly (Every 3 Months)

- [ ] **Review API key inventory**
  - Confirm all keys still active
  - Check for unused keys (delete)
  - Verify permissions haven't changed

- [ ] **Check rotation schedule**
  - Are any keys overdue?
  - Update due dates
  - Schedule reminders

- [ ] **Test all keys**
  ```bash
  # Run verification for each service
  bash ~/.openclaw/workspace/scripts/verify-tools-post-update.sh
  ```

- [ ] **Review audit logs**
  - Check for API errors (429, 403, 401)
  - Investigate rate limiting
  - Fix any authentication issues

### Annually (Every 12 Months)

- [ ] **Full security audit**
  - Review all API key permissions
  - Check for scope creep
  - Verify least-privilege principle
  - Remove unnecessary permissions

- [ ] **Rotate critical keys**
  - 1Password master password
  - Google Cloud service account
  - Any keys with "write" permissions

- [ ] **Update emergency procedures**
  - Test key revocation procedures
  - Verify backup/restore works
  - Document any process changes
