## 📋 Project Overview

### What is Onigashima?

Onigashima is a consumer-friendly distribution and hardware business built on top of OpenClaw and Momotaro iOS. It solves the problem: **"I want a personal AI assistant that knows me, respects my privacy, and doesn't require a PhD to set up."**

### Core Products

1. **Self-Install Tier** (Free → $99/year)
   - Visual installer for macOS
   - Freemium software model
   - Community support
   - DIY users, developers, enthusiasts

2. **Hardware Tier** ($899-$1,299)
   - Pre-configured Mac mini + OpenClaw + Momotaro
   - 1 year support + updates
   - Non-technical consumers
   - Business customers

3. **Cloud Services** (Included or $5/month)
   - Update management
   - Backup & restore
   - Analytics (private)
   - All users

### Market Opportunity

- **TAM:** 10M+ non-technical Mac users
- **SAM:** 2-5M willing to pay
- **SOM:** 50K-100K year 1 adopters
- **Year 5 Revenue Potential:** $12M-$60M

## 🛠️ Technical Architecture

### Installer Flow

```
User downloads Onigashima installer
    ↓
Opens visual installer (beautiful macOS UI)
    ↓
Step 1: System requirements check
    ↓
Step 2: Installation path selection
    ↓
Step 3: Configuration (network, API keys)
    ↓
Step 4: Automated installation
    ↓
Step 5: Verification suite runs
    ↓
Step 6: QR code for iPhone pairing
    ↓
✅ Installation complete (15 minutes)
```

### Cloud Infrastructure

- **User Management:** Database + auth service
- **Update Distribution:** S3 + CDN (fast delivery)
- **Backup Service:** Encrypted storage + restore
- **Analytics:** Private, aggregated data only
- **Monitoring:** System health + error tracking
- **Support Dashboard:** Ticket management + KB

### Security & Privacy

- Encryption at rest (user data)
- Encryption in transit (HTTPS/TLS)
- User control over data (delete anytime)
- Transparent privacy policy
- No data collection without consent
- Regular security audits
