# 🧠 Never-Forget Project Wiki: Core System

## 🎯 Project Overview
This project's core objective is to create and maintain a reliable, persistent, and structured cross-session task memory system. By integrating Obsidian as the single source of truth and protocolizing state handovers, we eliminate reliance on ephemeral chat context.
