# momo-kiji.dev

Landing page and marketing site for momo-kiji (CUDA for Apple Neural Engine).

## Development

```bash
npm install
npm run dev
```

Visit http://localhost:3000

## Deployment

Deployed on Vercel. Automatic deploys on push to main.

## Domain

momo-kiji.dev

## Setup

1. Install dependencies: `npm install`
2. Create `.env.local` with GA4 ID (if needed)
3. Deploy to Vercel or run locally
