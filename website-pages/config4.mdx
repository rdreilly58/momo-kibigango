# Config 4: Hybrid 3-Tier Speculative Decoding

## Local Speed Meets Cloud Quality

**92% Quality at 4% of the Cost**

Config 4 is a production-ready hybrid approach to speculative decoding that combines the speed of local inference with the quality of cloud APIs.

### The Problem

Most teams choose between speed and quality:
- **Pure local models:** Fast (15 tok/sec) but lower quality (70-85%)
- **Pure cloud APIs:** High quality (95%) but expensive ($15+/1000 requests) and slow (2-3 seconds)

There's no middle ground.

### The Solution

Config 4 handles both:
- **70% of requests:** Local models (instant, free)
- **30% of requests:** Claude Opus (quality, when needed)
- **Intelligent routing:** Semantic similarity scoring decides

### Architecture

```
REQUEST
  ↓
LOCAL PYRAMID (6s startup)
  ├─ Draft: Qwen 0.5B (ultra-fast)
  ├─ Qualifier: Phi-2 2.7B (fast filtering)
  └─ Score: Semantic similarity
  ↓
DECISION GATE
  ├─ Confidence ≥ 0.85 (70%) → ACCEPT (0.05s, $0)
  └─ Confidence < 0.85 (30%) → OPUS (2s, $0.015)
  ↓
RESPONSE
```

## Getting Started

### Installation

```bash
# Install dependencies
pip install anthropic sentence-transformers torch transformers

# Set API key
export ANTHROPIC_API_KEY="your-key"
```

### Quick Example

```python
from hybrid_pyramid_decoder import HybridPyramidDecoder

decoder = HybridPyramidDecoder()

result = decoder.generate("Write a haiku about code")

print(f"Source: {result['source']}")      # 'local' or 'opus_fallback'
print(f"Quality: {result['confidence']:.2f}")
print(f"Cost: ${result['cost']:.6f}")
print(f"Text: {result['text']}")
```

### View Statistics

```python
stats = decoder.get_stats()
print(f"Local acceptance: {stats['acceptance_rate_pct']:.1f}%")
print(f"Total cost: ${stats['total_cost']:.2f}")
```
