# Local Brother Printers

| Printer | Model | Type | Status |
|---------|-------|------|--------|
| **Brother_HL_L2350DW_series** | HL-L2350DW | Laser (B&W) | Online |
| **Brother_MFC_L2700DW_series** | MFC-L2700DW | MFP (B&W) | Online, Default |

## Quick Print
```bash
bash ~/.openclaw/workspace/skills/print-local/scripts/print-file.sh -f document.pdf
bash ~/.openclaw/workspace/skills/print-local/scripts/print-file.sh -f doc.pdf -p Brother_MFC_L2700DW_series -c 2 --duplex
```

## Capabilities
- Resolution: 2400 x 600 dpi
- Speed: 32-34 ppm
- Color: Black & white only
- MFC has scanner, copier, fax, ADF
