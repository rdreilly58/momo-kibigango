## Portfolio TypeScript Build Error Fixed (03:09 EDT)

### Problem
```
Type error: Type '{ children: Element[]; className: string; href: string | undefined; ... }' 
is not assignable to type 'LinkProps<any>'.
```

**Root cause:** Conditional component wrapper (Link vs <a>) had incompatible props.
- `<Link>` doesn't accept `target` and `rel` props
- Dynamic `Wrapper` component assignment confused TypeScript

### Solution
Replaced dynamic wrapper assignment with explicit if/else rendering:
```tsx
// Before (broken):
const Wrapper = isExternal ? "a" : Link;
<Wrapper href={...} target={...} rel={...} />  // ❌ Type mismatch

// After (fixed):
{isExternal ? (
  <a href={...} target="_blank" rel="noopener noreferrer" />
) : (
  <Link href={...} />
)}  // ✅ Each branch properly typed
```
