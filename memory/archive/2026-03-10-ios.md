## Technical Details

### GatewayClient Architecture
```
@MainActor class GatewayClient: ObservableObject
├─ Published: isConnected, connectionStatus, lastMessage, error
├─ connect(sessionToken) → URLSessionWebSocketTask
├─ receiveMessages() → async message loop
├─ sendCommand(text) → GatewayMessage
└─ attemptReconnect() → exponential backoff timer
```

### Message Flow
1. `sendCommand("text")` → Creates GatewayMessage
2. Encodes to JSON
3. Sends via URLSessionWebSocketTask
4. receiveMessages() loop listens for responses
5. Updates @Published properties → SwiftUI re-renders

### GatewayMessage (Codable)
```swift
struct GatewayMessage: Codable {
    let type: String
    let content: String
    let sessionId: String?
    let timestamp: String
}
```
