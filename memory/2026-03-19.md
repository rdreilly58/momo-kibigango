**Problem:** Links not visually obvious enough, needed better accessibility

**Solution Implemented:**
- All non-button links now underlined (WCAG accessibility standard)
- Nav links: Peach-colored underline (always visible, thicker on hover)
- Inline text links: White underline + peach background (high contrast)
- Footer links: White underline (always visible)
- Hover states: Thicker underlines + slight color/scale changes

**Impact:** 
- Clear visual signals that text is clickable
- Accessible for colorblind users (color + underline together)
- Professional, consistent design language
