### ✅ PDF Extraction from Emails (New Method)

**Step 1: Get thread ID**
```bash
gog gmail search 'subject:"search term"' --json | jq -r '.threads[0].id'
```

**Step 2: Read email content**
```bash
gog gmail thread get THREAD_ID
```

**Step 3: Download attachments**
```bash
cd /tmp && gog gmail thread attachments THREAD_ID --download --out-dir /tmp
```

**Step 4: Extract PDF text**
```bash
pdftotext /path/to/file.pdf - | head -200
```

**Complete pipeline:**
```bash
THREAD_ID=$(gog gmail search 'subject:SEARCH_TERM' --json | jq -r '.threads[0].id')
gog gmail thread get $THREAD_ID  # Read email
gog gmail thread attachments $THREAD_ID --download --out-dir /tmp  # Get PDF
```
