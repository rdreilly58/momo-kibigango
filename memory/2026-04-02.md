## Lessons Learned

1. **Multiple RC integrations = chaos.** Legacy standalone services + gateway plugin + RC webhooks all fighting. Clean up old approaches when migrating to new ones.
2. **Docker healthcheck: always test inside the container.** `curl` from host worked, but `curl` didn't exist inside the container. Use `docker exec` to verify.
3. **`localhost` doesn't always resolve in containers.** Use `127.0.0.1` explicitly.
4. **Plugin SDK version mismatches are silent killers.** Plugin connects and subscribes fine, but crashes on message handling. Check logs for "not a function" errors.
5. **Approval flow is painful for rapid debugging.** Many commands needed individual approval, slowing down the 1.5-hour debug session significantly.
